# Nurse Workflow

## Patient Check-In

1. Greet patient
2. Verify patient identity
3. Update contact information if needed
4. Check insurance status
5. Review allergy information
6. Document reason for visit
7. Assign to doctor/room

## Vital Signs Documentation

1. Obtain patient consent
2. Measure temperature
3. Measure blood pressure
4. Measure pulse rate
5. Measure respiratory rate
6. Document oxygen saturation
7. Record patient weight (if applicable)
8. Enter data into system
9. Flag critical values if needed

## Patient Monitoring

1. Check vital signs periodically
2. Document observations
3. Monitor for changes
4. Alert doctor of concerning changes
5. Provide patient comfort measures
6. Manage medications (if authorized)
7. Assist with procedures

## Specimen Collection

1. Verify patient identity
2. Explain collection procedure
3. Prepare supplies
4. Collect specimen per protocol
5. Label specimen correctly
6. Document collection details
7. Transport to lab
8. Track specimen status

## Discharge Planning

1. Provide discharge instructions
2. Explain medication instructions
3. Discuss activity restrictions
4. Schedule follow-up appointments
5. Provide written instructions
6. Answer patient questions
7. Document discharge
