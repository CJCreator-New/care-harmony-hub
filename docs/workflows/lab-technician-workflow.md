# Laboratory Technician Workflow

## Specimen Receiving

1. Receive specimen from collection area
2. Verify specimen label and details
3. Check specimen integrity
4. Verify collection tube type
5. Check collection time
6. Document receipt in system
7. Store at correct temperature
8. Track specimen movement

## Test Processing

1. Review lab order details
2. Prepare testing equipment
3. Calibrate equipment if needed
4. Follow test protocol
5. Process specimen
6. Record test results
7. Perform quality control checks
8. Verify results accuracy
9. Document processing details

## Result Entry

1. Obtain processed results
2. Enter values into system
3. Flag critical values
4. Verify entry accuracy
5. Document result entry time
6. Mark specimen as processed
7. Prepare report

## Quality Assurance

1. Perform daily quality checks
2. Run control samples
3. Verify equipment calibration
4. Review result patterns
5. Document QA activities
6. Address out-of-range results
7. Follow corrective actions

## Specimen Management

1. Monitor specimen storage
2. Follow stability guidelines
3. Discard expired specimens
4. Maintain proper temperatures
5. Track specimen location
6. Organize by test type
7. Maintain inventory

## Safety Protocols

1. Wear required PPE
2. Follow biohazard procedures
3. Use sharps containers correctly
4. Clean work area
5. Dispose of waste properly
6. Report incidents
7. Follow exposure protocols
