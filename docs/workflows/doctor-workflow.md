# Doctor Workflow

## Patient Consultation Workflow

1. Search for patient in system
2. Review patient history and current medications
3. Perform examination
4. Document findings
5. Create prescription if needed
6. Order lab tests if necessary
7. Provide patient with instructions
8. Save consultation notes

## Prescription Creation

1. Click "New Prescription"
2. Select patient
3. Search for medication
4. Check for interactions
5. Enter dosage and frequency
6. Add special instructions
7. Review and submit
8. Patient receives prescription

## Lab Order Management

1. Assess if lab testing needed
2. Click "Order Lab Test"
3. Select test type
4. Specify specimen type
5. Add clinical notes
6. Submit order
7. Track specimen collection
8. Review lab results when available
9. Communicate results to patient

## Referrals

1. Determine if specialist needed
2. Create referral order
3. Select specialist department
4. Add clinical notes
5. Send referral
6. Patient receives notification
7. Follow up on referral status
