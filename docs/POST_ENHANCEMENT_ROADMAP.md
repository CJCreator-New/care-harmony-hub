# CareSync HIMS — Post-Enhancement Roadmap

## Overview

This roadmap defines the priorities following the Phase 2 feature releases.

## Q3 2026

- Complete mobile app (Expo/React Native) feature parity with web
- Launch patient-facing mobile portal with appointment booking
- Automate RLS policy tests in CI pipeline

## Q4 2026

- AI clinical decision support — production rollout
- Predictive analytics for hospital resource planning
- Integration with external lab systems via HL7/FHIR

## 2027

- Multi-hospital federation (shared patient records with patient consent)
- Telemedicine video consultations (WebRTC integration)
- Insurance pre-authorisation workflow automation

## Maintenance Milestones

- Quarterly dependency updates
- Annual penetration test and security audit
- Continuous performance benchmarking (lighthouse, query profiler)

## Deprecation Notices

No APIs are currently scheduled for deprecation.
